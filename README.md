# Plasma-Payload_Controller
Microcontroller for SARP UW's experimental plasma actuator payload. Tasked with detecting engine shut-off, activation of payload, and logging.

# Required Libraries
 - <Adafruit_Sensor.h>
 - <Adafruit_LSM9DS0.h>
 - <Adafruit_BME280.h>
 - <SparkFunTMP102.h>
 - <Adafruit_Simple_AHRS.h>
 - <Adafruit_ADS1015.h>
